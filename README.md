📊 Sales Data Analysis using SQL

📌 Project Overview

This project focuses on analyzing a sales dataset using SQL to uncover key business insights. The goal is to transform raw transactional data into meaningful information that supports data-driven decision-making.

🎯 Problem Statement

Businesses generate large volumes of sales data, but without proper analysis, it becomes difficult to understand performance, customer behavior, and growth opportunities.

This project aims to answer critical business questions using SQL and derive actionable insights.

🗂 Dataset Information
- Database: sales_data_sample
- Type: Transactional Sales Data
- Key Columns:
              - Sales
              - Order Date
              - Country
              - Customer Name
              - Product Line
              - Order Status
